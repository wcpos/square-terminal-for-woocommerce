# Square Terminal for WooCommerce 0.2.2

This patch release restores the Square Terminal controls in WCPOS checkout. Recommended for all existing installs.

## Fixed

- **WCPOS payment UI rendering** — selecting Square Reader now displays the Terminal Device selector, Start Payment, and Check Status controls on the order-pay checkout. Cancel Payment appears after a Terminal payment starts.
- **Storefront checkout guard** — the interactive Terminal controls remain hidden before standard checkout creates an order, preventing payment requests with an invalid order ID.

## Upgrade notes

- No settings changes required. Requires PHP 8.1+, WordPress 6.5+, and WooCommerce 8.0+.
